# Password Strength Checker

## Overview
This project is a simple password strength checker developed using C/C++.

The program evaluates the strength of a password based on different security parameters such as length, use of uppercase letters, lowercase letters, numbers, and special characters.

## Features
- Checks password length
- Detects uppercase and lowercase characters
- Detects numbers
- Detects special characters
- Provides feedback on password strength

## Purpose
This project was developed to understand basic cybersecurity concepts related to password security and safe authentication practices.

## Technologies Used
- C / C++

## Status
Completed basic implementation. Future improvements may include stronger validation rules and enhanced feedback.
